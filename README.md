##Analisi Statistica Descrittiva – Mercato Immobiliare Texas
## Descrizione
Progetto di analisi statistica descrittiva su un dataset relativo al mercato immobiliare in Texas.

## Attività svolte
- Analisi esplorativa dei dati
- Studio delle distribuzioni delle variabili
- Calcolo indicatori statistici (media, varianza, ecc.)
- Visualizzazione tramite grafici

## Tecnologie
- R
- Analisi statistica

## Risultati
Individuazione di pattern e relazioni tra variabili del mercato immobiliare.
